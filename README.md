# libKitten
Collection of image and color calculations

## Installation
1. Download the latest `deb` from the [releases](https://github.com/Traurige/libKitten/releases)
2. Install libKitten

## Compatibility
iPhone, iPad and iPod running iOS/iPadOS 12 or later
